## ğŸš€ Quick Start

### Prerequisites
- Python 3.8+
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Microphone access for voice features

### Installation

1. **Clone or download the project**
   ```bash
   cd AI_voice
   ```

2. **Install Python dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up environment variables** (optional)
   Create a `.env` file in the root directory:
   ```env
   FLASK_ENV=development
   FLASK_DEBUG=1
   ```

4. **Run the web server**
   ```bash
   python web_server.py
   ```

5. **Open your browser**
   Navigate to `http://localhost:5000`

## ğŸ® Usage

### Voice Commands
- Click the microphone button to start voice recognition
- Speak clearly and wait for the assistant's response
- The assistant will respond both visually and audibly

### Text Input
- Type messages in the text input field
- Press Enter or click the send button
- Use Ctrl+K to quickly focus the input field

### Settings
- **Voice Speed**: Adjust speech synthesis speed (0.5x - 2.0x)
- **Voice Pitch**: Modify voice pitch (0.5x - 2.0x)
- **Auto-scroll**: Enable/disable automatic conversation scrolling
- **Theme**: Toggle between light and dark themes

### Keyboard Shortcuts
- `Ctrl+K`: Focus text input
- `Ctrl+M`: Toggle voice recognition
- `Ctrl+D`: Toggle theme
- `Enter`: Send message (when input is focused)

## ğŸ› Troubleshooting

### Voice Recognition Issues
1. **Check microphone permissions**: Ensure the browser has microphone access
2. **Use HTTPS**: Voice recognition requires a secure connection
3. **Browser compatibility**: Test with Chrome, Firefox, or Safari
4. **Clear browser cache**: Sometimes helps with permission issues

### Network Issues
1. **Check server status**: Ensure `web_server.py` is running
2. **Verify port**: Default port is 5000
3. **Firewall settings**: Ensure port 5000 is accessible
4. **CORS issues**: Check browser console for CORS errors

### Performance Issues
1. **Close other tabs**: Voice recognition can be resource-intensive
2. **Update browser**: Use the latest browser version
3. **Check system resources**: Ensure adequate RAM and CPU
4. **Disable extensions**: Some browser extensions can interfere
